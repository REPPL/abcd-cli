# Superseded

Intents killed by reclassification or absorption — folded into a larger intent or
replaced by a stricter successor. Each file records `superseded_by: <itd-M>` (the
successor) and `kind_at_supersession: <original-kind>` (the shape it had when
retired). Files here are preserved as historical record and never deleted.

See the [intents index](../README.md) for the corpus listing and the
[intent surface contract](../../brief/04-surfaces/05-intent.md) for the full
lifecycle.
