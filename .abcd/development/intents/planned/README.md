# Planned

Intents whose `/abcd:intent plan` has created a linked native spec stub — work is
committed but not yet shipped. Each intent here carries a bidirectional link to
its spec (`spec_id: fn-N`); bundle members share one spec with their bundle-mates.
An intent moves on to `shipped/` automatically when its linked spec closes.

See the [intents index](../README.md) for the corpus listing and the
[intent surface contract](../../brief/04-surfaces/05-intent.md) for the full
lifecycle.
