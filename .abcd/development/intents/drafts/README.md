# Drafts

Press-release intents captured but not yet promoted to a native spec — the bench
of forward-looking ideas, cheap to draft and discard. A standalone or
bundle-member intent leaves here for `planned/` once `/abcd:intent plan <itd-N>`
creates its linked spec stub; discipline-kind intents leave for `disciplines/`.
Directory location is the canonical lifecycle state.

See the [intents index](../README.md) for the corpus listing and the
[intent surface contract](../../brief/04-surfaces/05-intent.md) for the full
lifecycle.
