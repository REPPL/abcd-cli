# Interview context

The `interview` bounded context holds terms specific to the grill/interview
sub-verb of `/abcd:intent` — the interactive exchange (session) and its opening
move (embark). These pin vocabulary that means something narrower here than in the
core context.

Each term is a Markdown file with YAML frontmatter conforming to the terminology
schema. For the format specification, the validation command, and the full term
index, see the [glossary README](../README.md).
